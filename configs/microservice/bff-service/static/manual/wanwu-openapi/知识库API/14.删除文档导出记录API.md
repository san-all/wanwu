删除文档导出记录API
======

## 知识库删除文档导出记录API

知识库删除文档导出记录支持删除某个知识库下的某条导出记录。

### 请求接口

----

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | `http://localhost:8081/service/api/openapi/v1/knowledge/export/record` |
| 调用方式 | 同步调用                                                     |
| 字符编码 | UTF-8                                                       |
| 请求类型 | DELETE                                                       |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json`                             |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

----

| Parameter    | Required  | Type   | Instructions                     |
|---------------|----| ------ |--------|
| knowledgeId | 是  | string | 知识库Id  |
| exportRecordId   | 是  | string | 文档导出记录id  |




### 响应参数

----

| Parameter  | Type     | Instructions   |
|----------- |--------------| ---------- |
| code       | int          | 状态码        |
| data       | null         | 返回数据   |
| msg        | string       | 返回信息       |





### 请求示例

----

```shell
curl --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/knowledge/export/record' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' \
--data '{
  "exportRecordId": "2000764337534275584",
  "knowledgeId": "1998659550177136640"
}'
```

### 响应示例

----

```json
{
  "code": 0,
  "data": null,
  "msg": ""
}
```
