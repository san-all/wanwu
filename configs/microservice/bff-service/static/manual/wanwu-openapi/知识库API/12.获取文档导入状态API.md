获取文档导入状态API
======

## 知识库获取文档导入状态API
知识库获取文档导入状态API支持获取某个知识库下的最近的文件导入状态。


### 请求接口

----

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/import/tip` |
| 调用方式 | 同步调用                                                     |
| 字符编码 | UTF-8                                                       |
| 请求类型 | GET                                                       |
| 鉴权方式 | `Authorization: Bearer {API Key}`                         |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

----

| Parameter    | Required  | Type   | Instructions                     |
|---------------|----| ------ |--------|
| knowledgeId | 是  | string | 知识库Id  |






### 响应参数

----

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| code               | int          | 状态码        |
| data               | object         | 返回数据   |
| msg                | string       | 返回信息       |

#### data中详细字段

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| msg                | string       | 导入任务错误信息       |
| knowledgeId        | string       | 知识库id  |
| uploadstatus       | int          | 导入任务状态 0-任务待处理；1-任务导入中 ；2-任务完成；3-任务失败 |
| knowledgeName      | string       | 知识库名称  |





### 请求示例

----

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/import/tip?knowledgeId=1998659550177136640' \
--header 'Authorization: Bearer {API Key}' 
```

### 响应示例

----

```json
{
  "code": 0,
  "data": {
    "msg": "",
    "uploadstatus": 2,
    "knowledgeId": "1998659550177136640",
    "knowledgeName": "test"
  },
  "msg": ""
}
```
