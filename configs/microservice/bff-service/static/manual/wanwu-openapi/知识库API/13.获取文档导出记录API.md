获取文档导出记录API
======

## 知识库获取文档导出记录API
知识库获取文档导出记录支持获取某个知识库下的历史导出记录信息，可获取到导出文件的下载链接、导出时间、导出人、导出时间等信息。


### 请求接口

----

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/export/record/list` |
| 调用方式 | 同步调用                                                     |
| 字符编码 | UTF-8                                                       |
| 请求类型 | GET                                                       |
| 鉴权方式 | `Authorization: Bearer {API Key}`                         |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

----

| Parameter    | Required  | Type   | Instructions                     |
|---------------|----| ------ |--------|
| knowledgeId | 是  | string | 知识库Id  |
| pageSize    | 是  | int    | 分页大小  |
| pageNo      | 是  | int    | 分页序号  |





### 响应参数

----

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| code               | int          | 状态码        |
| data               | object         | 返回数据   |
| msg                | string       | 返回信息       |

#### data中详细字段

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| list               | object array | 文档详细信息       |
| docKnowledgeInfo   | object       | 知识库信息   |
| total              | int64        | 总数       |
| pageNo             | int          | 分页序号  |
| pageSize           | int          | 分页大小 |

#### list中详细字段

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| exportRecordId     | string       | 知识库导出记录id       |
| author             | string       | 导出人   |
| exportTime         | string       | 导出时间       |
| filePath           | string       | 导出文件下载路径  |
| status             | int          | 导出任务状态 0-任务待处理；1-任务导出中 ；2-任务完成；3-任务失败 |
| errorMsg           | string       | 导出状态错误信息  |
| knowledgeName      | string       | 知识库名称  |





### 请求示例

----

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/export/record/list?knowledgeId=1993492163547107328&pageSize=10&pageNo=1' \
--header 'Authorization: Bearer {API Key}' 
```
