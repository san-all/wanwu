获取文档配置API
======

## 获取文档配置API
获取文档配置支持获取某个知识库下某个文档解析的分段配置和解析方式。

### 请求接口

----

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/config` |
| 调用方式 | 同步调用                                                     |
| 字符编码 | UTF-8                                                       |
| 请求类型 | GET                                                       |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

----

| Parameter    | Required  | Type   | Instructions                     |
|---------------|----| ------ |--------|
| knowledgeId | 是  | string | 知识库Id  |
| docId    | 是  | int    | 文档id  |



### 响应参数

----

| Parameter  | Type     | Instructions   |
|--------------------|--------------| ---------- |
| code               | int          | 状态码        |
| data               | object         | 返回数据   |
| msg                | string       | 返回信息       |

#### data中详细字段

| Parameter  | Type     | Instructions   |
|---------------|--------------| ---------- |
| docImportType | int32 | 文档导入类型，0：文件上传，1：单条url上传，2.文件url上传    |
| docSegment    | object | 文档分段配置    |
| docAnalyzer   | string array | 文档解析类型 text / ocr  / model    |
| parserModelId | string | 模型解析或ocr模型id    |
| docPreprocess | string array | 文本预处理规则 replaceSymbols / deleteLinks    |



### 请求示例

----

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/config?knowledgeId=2005451236198846464&docId=2005522416335327232' \
--header 'Authorization: Bearer {API Key}'
```

### 响应示例

----

```json
{
  "code": 0,
  "data": {
    "docImportType": 0,
    "docSegment": {
      "segmentMethod": "0",
      "segmentType": "0"
    },
    "docAnalyzer": [
      "text"
    ],
    "parserModelId": "",
    "docPreprocess": [
      "replaceSymbols"
    ]
  },
  "msg": ""
}
```
