删除文档API
======

## 知识库删除文档API
知识库删除文档支持批量删除某个知识库下的若干文档。

### 请求接口

----

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc` |
| 调用方式 | 异步调用                                                     |
| 字符编码 | UTF-8                                                       |
| 请求类型 | DELETE                                                      |
| 鉴权方式 | `Authorization: Bearer {API Key}`                           |
| 请求格式 | `Content-Type: application/json`                            |
| 响应格式 | `Content-Type: application/json`                            |

### 请求参数

----

| Parameter    | Required  | Type   | Instructions|
|---------------|----| ------ |--------|
| knowledgeId | 是  | string | 知识库Id  |
| docIdList   | 是  | string array | 文档id列表  |




### 响应参数

----

| 参数名              | 类型           | 说明         |
|--------------------|--------------| ---------- |
| code               | int          | 状态码        |
| data               | null         | 返回数据   |
| msg                | string       | 返回信息       |





### 请求示例

----

```shell
curl --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/knowledge/doc' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' \
--data '{
  "docIdList": [
    "2005529991252545536"
  ],
  "knowledgeId": "2005451236198846464"
}'
```

### 响应示例

----

```json
{
  "code": 0,
  "data": null,
  "msg": ""
}
```
