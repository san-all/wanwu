# 查询知识库列表API
根据名称、标签等条件查询知识库列表

## 接口信息

| 项目 | 说明 |
|------|------|
| **请求方法** | POST |
| **请求URL** | `http://localhost:8081/service/api/openapi/v1/knowledge/select` |
| **字符编码** | UTF-8 |
| **请求格式** | `Content-Type: application/json` |
| **响应格式** | `Content-Type: application/json` |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}`                        |


## 请求参数（JSON Body）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `name` | String | 否 | 知识库名称（支持模糊查询） |
| `tagId` | Array[String] | 否 | 标签ID列表，用于筛选指定标签的知识库 |


## 响应参数

| 状态码 | 说明 | 响应体结构 |
|--------|------|------------|
| 200 | 成功 | [Response](#response-对象) |

### Response 对象
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `code` | Integer | 状态码（0表示成功） |
| `msg` | String | 提示信息 |
| `data` | [KnowledgeListResp](#knowledgelistresp-对象) | 响应数据，包含知识库列表 |

### 子对象结构说明
#### KnowledgeListResp 对象
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `knowledgeList` | Array[[KnowledgeInfo](#knowledgeinfo-对象)] | 知识库列表数据 |

#### KnowledgeInfo 对象
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `knowledgeId` | String | 知识库ID |
| `name` | String | 知识库名称 |
| `orgName` | String | 知识库所属组织名称 |
| `description` | String | 知识库描述 |
| `docCount` | Integer | 知识库内文档数量 |
| `embeddingModelInfo` | [EmbeddingModelInfo](#embeddingmodelinfo-对象) | 嵌入模型信息 |
| `knowledgeTagList` | Array[[KnowledgeTag](#knowledgetag-对象)] | 知识库标签列表 |
| `createUserId` | String | 创建人ID |
| `createAt` | String | 创建时间（格式参考业务实际返回值，如YYYY-MM-DD HH:mm:ss） |
| `permissionType` | Integer | 权限类型：<br>• `0` - 查看权限<br>• `10` - 编辑权限<br>• `20` - 授权权限（数值不连续为预留中间权限，优先级：授权权限>编辑权限>查看权限） |
| `share` | Boolean | 知识库权限类型：<br>• `true` - 共享<br>• `false` - 私有 |
| `ragName` | String | RAG名称 |
| `graphSwitch` | Integer | 知识图谱开关（无具体枚举说明则按业务实际） |
| `category` | Integer | 知识库类型：<br>• `0` - 知识库<br>• `1` - 问答库 |
| `llmModelId` | String | 知识图谱模型ID |
| `updatedAt` | String | 更新时间（格式参考业务实际返回值，如YYYY-MM-DD HH:mm:ss） |

#### EmbeddingModelInfo 对象
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `modelId` | String | 嵌入模型ID |

#### KnowledgeTag 对象
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `tagId` | String | 知识库标签ID |
| `tagName` | String | 知识库标签名称 |
| `selected` | Boolean | 该标签是否被选中 |


## 请求示例

```json
{
    "name": "",
    "tagId": []
}
```

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/select' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' \
--data '{
    "name": "",
    "tagId": []
}'
```
